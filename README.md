# AlQuran

AlQuran is an app that allows you to listen to chapters of the Quran, voiced by different reciters. It also supports Android Auto.

## Features

- Listen to chapters of the Quran, voiced by different reciters.
- Support for Android Auto.
- Offline playback.
- Pause and resume playback.
- Seek to a specific position in the chapter.
- Choose your preferred reciter.

## API
The app uses the [MP3Quran API](https://www.mp3quran.net/eng/api) to fetch the Quran chapters and reciters. I would like to praise the developers of this API for their great work and making Quran accessible to everyone.
